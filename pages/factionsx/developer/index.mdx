# Developer Guide

Welcome to the developer guide for FactionsX.

* **Addons**: FactionsX uses addons which are loaded from it's own classloader and can be considered 
"plugins" for FactionsX in the same way FactionsX is a "plugin" for spigot.
    * **[Addon Creation Guide](/factionsx/developer/addons/)**: learn how to create your own factionsx addons.
